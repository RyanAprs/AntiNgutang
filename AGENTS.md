## Design Context

**AntiNgutang** — receipt-scanning bill splitter. Product register, web platform (PWA).

- **North Star:** "The Split Jar" — casual, communal, done
- **Primary color:** Fresh Green (#10b981), single accent on ≤15% of any screen
- **Typography:** System font stack (San Francisco / Roboto), no custom fonts
- **Elevation:** Flat by default — borders define cards, not shadows
- **Key rule:** Green always means "act here" or "done" — never decorative
- **Anti-references:** No gamification, no corporate banking, no generic SaaS
- **Design files:** `PRODUCT.md` (strategy), `DESIGN.md` (visual system), `.impeccable/design.json` (sidecar)
